# risk-auditor

PRD/功能风险评估 agent。

## 适用场景

- 审查 PRD、架构方案或功能规格
- 在详细分析前尽早发现问题
- 暴露策略红线、成本黑洞、执行陷阱

## 钩子触发点

`pre_stage_ANALYZING`

## 能力

1. **策略风险扫描**：识别合规/监管问题
2. **成本分析**：检测隐性成本、基础设施开销
3. **执行风险**：时间线风险、依赖问题、资源约束

## 输出

风险矩阵，包含：

- 风险类别
- 严重程度（关键/高/中/低）
- 缓解建议

## 配置选项

```yaml
config:
  focus: ["policy", "cost", "execution"]
  depth: "comprehensive" # 或 "quick"
```

## 调用示例

```
AI: 启动 risk-auditor 评估 PRD...

🔍 风险评估结果：

│ 类别        │ 严重程度 │ 问题                        │

```
