# Wiki page templates

All wiki page templates used by the llm-wiki skill (entity-first layout).

> Conventions: every page starts with an `# Title` H1; cross-references use `[[folder/slug]]` (cap ~5, use "See also" beyond that);
> slugs are kebab-case; source pages carry `**Source**:` / `**Ingested**:`.

---

## Entity types (wiki/people, organizations, places, products, events, concepts, other-entities)

### Person (wiki/people/)

```markdown
# {Name}

**Role**: {one-line identity}
**Aliases**: {optional}

## Overview
{description synthesized from all sources}

## Key facts
- {date} {fact} (source: [[threads/{src}]] / [[articles/{src}]])

## Relationships
- → {relationship} → [[organizations/{org}]], [[people/{other}]]

## See also
- [[products/{proj}]]
```

### Organization (wiki/organizations/)

```markdown
# {Organization / team name}

**Type**: {company | team | community}

## Overview
{description}

## Members / affiliations
- [[people/{person}]] — {role}

## Key facts
- {date} {fact} (source: [[...]])
```

### Place (wiki/places/)

```markdown
# {Place name}

**Type**: {city | venue | region}

## Overview
{description}

## Affiliations
- [[events/{event}]], [[organizations/{org}]]
```

### Product / project (wiki/products/)

```markdown
# {Product / project name}

**Type**: {product | project | initiative}
**Status**: {active | shipped | sunset}

## Overview
{what it is, who owns it}

## Progress
- {date} {milestone / number} (source: [[threads/{src}]])

## Affiliations
- Owner [[people/{owner}]]; part of [[organizations/{org}]]
```

### Event / decision (wiki/events/)

```markdown
# {Event / decision title}

**Date**: {date}
**Type**: {decision | launch | incident | milestone}
**People involved**: [[people/{who}]]

## Background
{what led to this event / decision}

## Content
{what happened / what was chosen and why}

## Consequences
- {positive / negative impact}

## See also
- [[products/{proj}]], [[concepts/{topic}]]
```

### Concept / term / code pattern (wiki/concepts/)

```markdown
# {Concept name}

**Discipline**: {optional, used for freshness classification}
**Verified**: {no — for easily-stale technical content}

## Definition
{a clear definition synthesized from all sources}

## Key sources
- [[articles/{source1}]] — {what that source says}
- [[podcasts/{source2}]] — {supplement}

## Code / usage (if it is a code pattern)
{code block with a language tag + when to use it + common pitfalls}

## Related concepts
- [[concepts/{related}]] — {relationship}

## Evolution
| Date | Update | Source |
|------|------|------|
| {date} | Created from {source} | [[articles/{source}]] |
```

## Source types (wiki/articles, podcasts, vlogs, diaries, threads)

### Article / podcast / video (wiki/articles · podcasts · vlogs/)

```markdown
# {Source title}

**Source**: [[raw/{articles|podcasts|vlogs}/{slug}]] or {URL}
**Ingested**: {date}
**Last verified**: {date}
**Author**: {author} | **Year**: {year}

## Key points
- {point 1}
- {point 2}

## Concepts / entities referenced
- [[concepts/{c}]], [[people/{p}]]

## Contradictions with existing knowledge
- {conflict, or "none detected"}
```

### Diary / chronological stream (wiki/diaries/)

```markdown
# {ISO week, e.g. 2026-W22}

## 2026-MM-DD
- **#{group/scene}** {person} {time}: {brief summary (keep key numbers / links / decisions)} ([[people/{p}]] / [[products/{proj}]])
```

### Conversation consolidation (wiki/threads/)

```markdown
# {Group / conversation name}

**Source**: [[raw/lark/{alias}-{date}]]
**Chat**: {alias}

## Overview
{what this group discusses, its long-running themes}

## Recent consolidations
- {date} {topic / decision} ([[events/{e}]], [[people/{p}]])
```
