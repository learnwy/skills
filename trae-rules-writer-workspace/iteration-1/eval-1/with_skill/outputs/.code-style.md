# Code Style

## Naming Conventions

- **Components**: Use PascalCase (e.g., `UserProfile`, `ButtonGroup`)
- **Functions/Hooks**: Use camelCase (e.g., `handleClick`, `useAuth`)
- **Constants**: Use UPPER_SNAKE_CASE (e.g., `MAX_RETRY_COUNT`)
- **Interfaces**: Use PascalCase with optional I prefix (e.g., `User`, `IUser`)
