# Personal Skills Repository

## 🎯 项目

个人 Skills 仓库，存放 AI 技能。

## 📁 结构

```
skills/
├── .trae/rules/
├── skills/
│   ├── life-reset/
│   └── software-methodology-toolkit/
```

**agents/** - 方法论（problem-definer, story-mapper）

**skills/** - 能力模块
- 项目特定（高优先级）
- 通用（Fallback）
